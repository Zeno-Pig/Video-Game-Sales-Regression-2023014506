# Video-Game-Sales-Regression-2023014506
# 回归分析第一次实验报告
# 电子游戏全球销量影响因素回归分析

本项目通过对 16,000+ 条电子游戏销售数据进行多元线性回归分析，探讨了**发行年份、游戏平台、游戏类型**对全球销量（Global Sales）的显著性影响。

## 核心发现
- **R² = 0.121**：模型解释了销量波动的 12.1%，显示游戏销量受爆款效应等不可见因素影响显著。
- **年份效应**：单款游戏平均销量随时间推移呈微弱下降趋势（系数 -0.0199）。
- **优势品类**：平台类 (Platform) 和 射击类 (Shooter) 展现出显著的正向销量贡献。

## 技术栈
- **语言**：Python 3.9
- **核心库**：`statsmodels` (统计建模), `pandas` (数据处理), `matplotlib/seaborn` (可视化)
